# HailTrack CRM - White-Label Edition

**The only field sales CRM built specifically for auto hail damage repair shops.**

## Quick Start

```bash
pnpm install
pnpm db:push
pnpm dev
```

Visit http://localhost:3000

## Features

- Offline-first architecture
- GPS tracking & route optimization
- Kanban pipeline management
- SMS & push notifications
- Photo documentation with OCR
- Multi-tenant ready
